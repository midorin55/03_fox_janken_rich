## （1）制作サイトのタイトル
-東八拳じゃんけん
## （2）工夫した点・こだわった点
-日本のじゃんけんのルーツである江戸時代の東八拳でじゃんけんを作成しました。
-最初に初心者でも分かりやすいようシンプルなルール説明を加えました
-可愛い絵を加える事で見た目も飽きさせないような工夫をしました
-前回提出時よりも更に、遊ぶ時の楽しさを盛り上げるため、「勝負はじめ」「狐」「庄屋」「鉄砲」などのボタン選択時に音声を付け加えるようにしました

## （3）難しかった点・次回トライしたいこと（または機能）
-条件分岐を作成する際に、いきなりVS上で作成するのは難しくメモや分岐を紙で整理しながら作成。ただし、それでも勝っているのに負け判定になったり条件分岐がどこか間違えている等発生＆ズレが生じたりして、チャッピーと相談してたどり着けました。
-本来の理想の姿はチャッピーに修正してもらった部分、正解部分を最終的に自分で何も見なくても導ける、書ける事なので、頑張りたいです。
-どこかのアプリ作成のところで自作の絵を組み込んでみたいです
